## 📱 Uygulama Hakkında

**UniPazar**, Türkiye'deki üniversite öğrencilerinin ihtiyaç duymadıkları ürünleri **kampüs içinde güvenli, hızlı ve kolay** bir şekilde alıp satabilecekleri, öğrenciye özel bir Android ikinci el pazaryeri uygulamasıdır.

> 🏫 **İstanbul Okan Üniversitesi** — Mobil Teknolojileri Programı Mezuniyet Projesi

---

## ✨ Özellikler

| Özellik | Açıklama |
|---------|----------|
| 🔐 **Kimlik Doğrulama** | Firebase ile e-posta/şifre girişi, şifre sıfırlama |
| 🏷️ **İlan Yönetimi** | Fotoğraflı ilan ekleme, kategorilere ayırma, satıldı işaretleme |
| 🏫 **Üniversite Filtresi** | 200+ Türk üniversitesine göre ilan filtreleme |
| 💬 **Gerçek Zamanlı Mesajlaşma** | Anlık mesajlaşma, fotoğraf gönderme, okundu bilgisi |
| 🔔 **Bildirim Sistemi** | Firebase Cloud Messaging ile push bildirimler |
| 👤 **Profil Yönetimi** | Profil fotoğrafı, biyografi, üniversite güncelleme |
| 🛡️ **Güvenlik** | Kullanıcı engelleme ve şikayet mekanizması |
| 💰 **Teklif Verme** | Fiyat pazarlığı için özel teklif gönderme |

---

## 🛠️ Kullanılan Teknolojiler

- **Dil:** Kotlin 2.0.0
- **IDE:** Android Studio Ladybug
- **Min SDK:** Android 8.0 (API 26)
- **Backend:** Firebase (Auth, Firestore, Storage, FCM)
- **Görseller:** Glide 4.16.0
- **UI:** Material Design 3, Shimmer animasyonu

---

## 📊 Proje İstatistikleri

- 📁 **32** Kotlin dosyası
- 🎨 **29** XML layout dosyası
- 📝 **~4.144** satır kod
- 🏫 **200+** Türk üniversitesi desteği
- 📱 **16.6 MB** APK boyutu
